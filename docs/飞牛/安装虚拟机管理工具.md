所有安装全部进入飞牛进行安装，不是在宿主机上安装！

# VMware、ESXI

```bash
    sudo apt update
    sudo apt install open-vm-tools
```

# PVE Qemu

```bash
    sudo apt update
    sudo apt install qemu-guest-agent -y
```

安装后记得重启飞牛即可，不出意外VMware管理台可以正确识别到虚拟机并且支持系统层面软关机等。
